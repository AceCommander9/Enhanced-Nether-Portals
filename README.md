A Minecraft datapack + mod that introduces unique ruined Nether portals with small, self-contained miniature Nether biomes spawning around them.
Each portal generates with custom environmental details, Nether-like terrain, ambience, and structural variations that make Overworld exploration richer and more mysterious.
